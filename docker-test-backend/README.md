## Simple commands to create a very small Node.js backend to test Docker.

---

### 1. Create a New Project

```bash
mkdir docker-test-backend
cd docker-test-backend
npm init -y
```

---

### 2. Install Express

```bash
npm install express
```

### 3. Run Server Locally (Optional Test)

```bash
node app.mjs
```

Then check:
[http://localhost:3000](http://localhost:3000)
